# IooBot
一个适用于[OPQBOT](https://github.com/OPQBOT/OPQ) 的识图姬  
有一些奇奇怪怪的功能
如果有什么BUG或者什么建议可以通过机器人QQ **850558946**使用命令#留言 <内容> 向我反馈（**#help**查看命令）
目前已经实现插件化，部分插件可拆下搭载在其他机器人上（同为botoy）使用  
**一般每周六定期维护**  
使用见  →[wiki](https://github.com/kitUIN/ioobot/wiki) ←
